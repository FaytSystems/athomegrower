Image policy.

The safest long-term plan is owned photography. This starter package uses real publishable reference images where available, source links, and visible callout overlays.

Rules

  • Use owned photos first.
  • Use public-domain or clearly licensed photos second.
  • Credit image sources on-page or in the image credits page.
  • Store annotations separately from originals when moving to a database.
  • Use text alternatives for every important photo callout.
